Quick Start

Install the package and initialize a gradient in just a few lines:

import { NeatGradient } from "@firecms/neat";

const canvas = document.getElementById("gradient");
const gradient = new NeatGradient({
    ref: canvas,
    colors: [
        { color: "#FF5572", enabled: true },
        { color: "#4CB4BB", enabled: true },
        { color: "#FFC600", enabled: true },
    ],
    speed: 4,
    horizontalPressure: 3,
    verticalPressure: 4,
});

// React to scroll
window.addEventListener("scroll", () => {
    gradient.yOffset = window.scrollY;
});

Designed for the Modern Web

Neat renders on a standard <canvas> element using raw WebGL. It works everywhere — React, Vue, Svelte, plain HTML — with no framework lock-in.

The gradient adapts to any container size via a built-in ResizeObserver, and supports high-DPI displays out of the box.
